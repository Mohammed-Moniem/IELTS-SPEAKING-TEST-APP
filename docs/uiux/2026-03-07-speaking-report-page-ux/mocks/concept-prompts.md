# Mock Prompt Set

## Concept 1 Prompt

Create a web speaking-report page for an IELTS SaaS product using a clean white canvas, strong violet accent color, and premium but restrained spacing. The layout should feel like a coaching debrief, not a dashboard. Above the fold, show the question prompt, overall band, a short verdict sentence, session date, and one strong CTA such as `Retry this prompt`. Below, show `Why this score happened`, a compact transcript block that adapts to short answers, rubric score tiles, strengths, improvements, and targeted next actions. Avoid giant empty transcript panels, avoid metadata-heavy headers, and avoid generic analytics styling.

## Concept 2 Prompt

Design an analytical speaking-report page with an adaptive transcript column and a sticky insights rail. Keep the current Spokio visual language with violet accents and rounded cards, but make the experience feel fuller and more useful. The right rail should contain score, summary, and next steps. The left side should contain transcript and explanation. Handle short transcripts gracefully without leaving large blank areas.

## Concept 3 Prompt

Design a speaking-report page that ends in an improvement plan. The page should begin with a concise result summary, then show why the score happened, then recommend the next best practice action based on weak skill areas. Use clear typography, calm spacing, and explicit action buttons. Avoid decorative filler and avoid empty-state language that feels like missing analysis.

## Constraints

- Keep current brand cues unless explicitly changed.
- Include web and mobile variants where relevant.
- Do not redesign the whole app shell.
- Make the page feel higher value than the inline review, but consistent with it.
