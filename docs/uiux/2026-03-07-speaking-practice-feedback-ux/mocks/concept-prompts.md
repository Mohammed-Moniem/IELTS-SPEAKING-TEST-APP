# Mock Prompt Set

## Concept 1 Prompt

- Design a desktop-first speaking practice screen for Spokio IELTS SaaS. Keep the existing violet brand accents and clean, light interface. Show one large session card that contains the prompt, recording controls, a prominent processing status block, then an inline review summary with band score, transcript preview, strengths, improvements, and a secondary `Open full report` button. Avoid separate low-priority cards below the fold for transcription and score.

## Concept 2 Prompt

- Design a speaking practice desktop layout with a left session workspace and a sticky right review rail. The left side contains the prompt, microphone selector, recording controls, and fallback manual transcript. The right rail shows processing progress and then transforms into a compact feedback summary. Keep the current Spokio brand palette and avoid dark-mode visuals.

## Concept 3 Prompt (optional)

- Design a speaking practice result-takeover view for Spokio. After the learner uploads a recording, the recorder panel transitions into a focused result view with band score, transcript, top strengths, top improvements, and next-step CTAs. Preserve the current soft violet/light-gray brand system and avoid generic AI-assistant styling.

## Constraints
- Keep current brand cues unless explicitly changed.
- Include web and mobile variants where relevant.
- Keep the details page as a secondary drill-down, not the primary first review surface.
- The first feedback state must be visible without a long scroll on a standard desktop viewport.
